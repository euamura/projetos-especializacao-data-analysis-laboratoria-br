# Projetos — Especialização em Data Analysis · Laboratória Brasil

Repositório com os projetos desenvolvidos durante o **Bootcamp de Especialização em Data Analysis** da [Laboratória Brasil](https://www.laboratoria.la/br).

Cada pasta contém um projeto independente, com documentação, arquivos de análise e links para dashboards e apresentações externas.

---

## Projetos

| Projeto | Tema | Ferramentas |
|---|---|---|
| [amazon-sales](./amazon-sales/) | Análise de produtos para estratégias de dropshipping | BigQuery · Power BI |
| [analise-spotify](./analise-spotify/) | Validação de hipóteses sobre sucesso musical | BigQuery · Power BI · Python |
| [risco-relativo-aplicado-a-inadimplencia](./risco-relativo-aplicado-a-inadimplencia/) | Risco de crédito e score de inadimplência | BigQuery · Python · Looker Studio |
| [segmentacao-mercado-rfm](./segmentacao-mercado-rfm/) | Segmentação de clientes por modelo RFM | BigQuery · Google Sheets |
| [superstore](./superstore/) | Pipeline ETL e modelagem dimensional | BigQuery · Power BI · Python |
